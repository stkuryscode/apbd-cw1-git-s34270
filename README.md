Aplikacja wczytuje tablicę cyfr od użytkownika i wypisuje je słownie.
Dodano liczenie średniej
Historie obu gałęzi się rozeszły, więc Git musiał utworzyć specjalny commit
Dodano liczenie wartosci maksymalnej
1.Fast-forward dzieje się wtedy, gdy główna gałąź nie miała nowych commitów i przesuwa ją do przodu o nowe zmiany, a
merge commit powstaje gdy na obu łączonych gałęziach działo się coś niezależnie i Git musi stworzyć zupełnie nowy commit żeby je połączyć

2.Rebase bierze nasze commity i dokleja je na samym końcu głównej gałęzi, a merge zachowuje pełną historię rozgałęzień tworząc nowy commit

3.Usunąłem komunikaty o konflikcie i ustawiłem jedną wersje komunikatu.
